Administrar usuarios

Para aprobar un usuario debe dejar vacío el campo Llave de registro, si el usuario está pendiente el campo mostrará pending. Si desea bloquear al usuario debe escribir en el campo blocked.

El campo Requires Approval indica si éste usuario necesitará que cada publicación que realice necesite ser aprobada para que se publique. Si no quiere aprobar cada publicación deberá marcar la casilla.

Is Admin indica si éste usuario tiene permisos de administración en la aplicación.
